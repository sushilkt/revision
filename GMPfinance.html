Financial Ratio

  1. liquidity measure
    • current ratio : current asset/ current liability ()
    • acis-test: quick ratio : current asset - inventory / current liability
    • interval measure : current asset / average daily operating cost :(low long business can run in crisis like corona )
  2. Long term solvency measure / levarage ration / financial levarage ratio
    • total debt ratio : total asset-total equity / total asset
    • debt-equity ratio : total debt / total equity
    • long term debt ratio : long term debt / long term debt +total equity
    • time interest ratio : EBIT/Interest
    • cash coverage ratio : EBITDA/Interest
  3. Asset managment/ Turnover ratio /
    • inventory turnover : COGS/Inventory
    • days sale in inventory: 365/inventory turnover
    • recievable turnover : sales/account recievable
    • Days sales recievebale: 365/recievebale turnover
    • asset turnover:
      • NWC turnover : Sales/Net wroking capital
      • total asset turnover : Sales/Total asset
  4. profitability measure
    • profit margin : net income/sales
    • return on asset (ROA): net income/total sset
    • return on equity(ROE):net income/total equity
  5. Market value Measure
    • EPS: net income/Shares Outstanding
    • PE ratio :price per share / Earning per share
    • market to book ratio: market vaue per share / book value per share
